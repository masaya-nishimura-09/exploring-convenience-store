# 75

## 概要

| 設定 | 経路                                                 | Q 更新法      | パラメータ                               | 商品判別   | 購入ルール                                                                         | 追加事項         |
| ---- | ---------------------------------------------------- | ------------- | ---------------------------------------- | ---------- | ---------------------------------------------------------------------------------- | ---------------- |
| 75   | ATM → 朝食、飲み物、おやつをそれぞれ 1 つずつ → レジ | 並列 Q 更新法 | ε = 0.01, Map1, similarity_threshold=0.9 | 写真で判別 | 最安値商品のみ購入。所持金不足なら購入しない。店内写真と買い物リストの写真は違う。 | マップ範囲外に行くとq値を下げる。最大歩数を設定。prefer_lower_price_itemsを削除 |

## 設定ファイルについて(./config.json)

- version: このプロジェクトの番号
- item_amount: ./input/shopping_list 内の写真の枚数（実行後に自動作成）
- epsilon: ε の値
- starting_x: 開始地点の x 座標
- starting_y: 開始地点の y 座標
- show_output:結果を都度ターミナルに出力するときは true
- similarity_threshold: 類似度のしきい値
- title: 実験の手法
- runs_per_set: 1 セットあたりの実行回数,
- sets: セット数

## 前回からの変更点

- 最大歩数を設定し、それを超えるとレジに移動
- prefer_lower_price_itemsは使わないので削除

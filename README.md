# exploring_convenience_store

## 3 つの手法について

| 名前               | 更新法                                        | 使用及び比較法                                                                            |
| ------------------ | --------------------------------------------- | ----------------------------------------------------------------------------------------- |
| 直列 Q 更新法      | 使用している q マップのみ更新                 | 使用している q マップで次の進行方向を決定                                                 |
| 直列+並列 Q 更新法 | 商品を探している間は全ての商品 q マップを更新 | 商品を探している間は全ての商品 q マップ(まだ見つけていない)を比較して次の進行方向を決定。 |
| 並列 Q 更新法      | 全ての q マップを更新                         | 商品を探している間は全ての商品 q マップ(まだ見つけていない)を比較して次の進行方向を決定。 |

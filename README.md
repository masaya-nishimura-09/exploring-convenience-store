# exploring_convenience_store

## 概要

| 設定 | 経路                                  | Q 更新法      | パラメータ     | 商品判別   | 購入ルール                                                                                                         |
| ---- | ------------------------------------- | ------------- | -------------- | ---------- | ------------------------------------------------------------------------------------------------------------------ |
| 61a  | ATM -> (x, y, z) -> レジ              | 並列 Q 更新法 | ε = 0.01, Map1 | 写真で判別 | レジで会計し、最安値の商品だけを購入する。また、所持金が足りなければ買わない。                                     |
| 61b  | ATM -> (x, y, z) -> レジ              | 並列 Q 更新法 | ε = 0.01, Map1 | 写真で判別 | 最初に見つけた商品のみを購入する(一つ目の商品を見つけるとレジに行く)。レジで会計し、所持金が足りなければ買わない。 |
| 66   | ATM → 水(x と y の 2 種類) → レジ     | 並列 Q 更新法 | ε = 0.01, Map1 | 写真で判別 | 最安値商品のみ購入。所持金不足なら購入しない                                                                       |
| 67   | ATM → 水(2 種類), 弁当(2 種類) → レジ | 並列 Q 更新法 | ε = 0.01, Map1 | 写真で判別 | 最安値商品のみ購入。所持金不足なら購入しない                                                                       |
